⚡
Cursor
AIコードエディター
  1. 1cursor.com にアクセスしてダウンロードボタンをクリック
  2. 2ダウンロードした .dmg（Mac）または .exe（Windows）を開いてインストール
  3. 3Cursorを起動し、アカウントを作成またはGoogleでサインイン
  4. 4初期設定のウィザードに沿って完了させる
🤖
Claude Code
AnthropicのAI CLIツール
  1. 1ターミナル（Mac）またはコマンドプロンプト（Windows）を開く
  2. 2以下のコマンドを入力してインストール：
    npm install -g @anthropic-ai/claude-code
  3. 3claude と入力してEnterで起動を確認
  4. 4Anthropicアカウントでログインし、APIキーを設定
📊
Google スプレッドシート
Googleのオンライン表計算ツール
  1. 1Googleアカウントにログイン（持っていない場合は作成）
  2. 2Googleドライブ（drive.google.com）を開く
  3. 3「＋ 新規」→「Googleスプレッドシート」をクリックして作成
  4. 4ブックマークしておくと次回以降スムーズ
🌐
Claude in Chrome
Chrome拡張機能でClaudeを使う
  1. 1Google Chromeを開く（インストールしていない場合は先にインストール）
  2. 2Chrome ウェブストアで「Claude」を検索
  3. 3「Chromeに追加」をクリックして拡張機能をインストール
  4. 4Anthropicアカウントでログインして完了
✅
Addness
タスク・ゴール管理ツール
セットアップ手順
  1. 1下のボタンからサインアップページを開く
  2. 2「Googleでログイン」をタップしてアカウントを選択
  3. 3ログイン完了でセットアップ終了
→ Addness サインアップはこちら
